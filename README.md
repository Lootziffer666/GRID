## 🧭 Was ist PAINKILLER?

Android-Utility-Suite die wiederkehrende Smartphone-Schmerzpunkte löst. Keine Bloatware, kein Tracking, keine Werbung.

---

## 🏗 Architektur

```
PAINKILLER/
├── app/       # Compose UI
├── domain/    # Business Logic (pure Kotlin)
├── handoff/   # Agent-Doku
└── templates/
```

**Stack:** Kotlin · Compose · Clean Architecture · Material 3 · Hilt · Coroutines + Flow
